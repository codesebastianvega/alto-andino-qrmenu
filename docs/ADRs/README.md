# Architecture Decision Records (ADR)

Este directorio contiene los registros de las decisiones arquitectónicas clave tomadas en el proyecto Aluna. El objetivo es documentar no solo **qué** tecnología usamos, sino el **porqué** y qué alternativas consideramos.

## 📄 Lista de ADRs

*   **[0000-template.md](file:///d:/BUSSINESS%20CHAN/CHAN/Programacion/alto-andino-qrmenu/docs/ADRs/0000-template.md)**: Plantilla para nuevos registros.
*   *Próximamente: ADR 0001 (Supabase RLS), ADR 0002 (Vite Stack).*

## 🛠️ Cómo crear un ADR
1. Copia el contenido de `0000-template.md`.
2. Crea un nuevo archivo siguiendo la numeración (ej. `0003-my-decision.md`).
3. Describe el contexto, las opciones y la decisión final.
